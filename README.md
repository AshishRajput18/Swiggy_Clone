# Swiggy Clone

A React-based Swiggy clone web application that mimics popular features of Swiggy, an online food delivery platform. This project demonstrates UI components, state management, and basic routing in React.

## Features

- **Header Component:** Search, Offers, Help, Sign In, Cart.
- **Top Restaurants:** Display top restaurants with ratings.
- **Best Cuisines & Places:** Showcase popular cuisines and places nearby.
- **Category Section:** Browse food by categories.
- **Online Delivery Section:** Explore restaurants offering online delivery.
- **Responsive Layout:** Works on desktop and mobile screens.
